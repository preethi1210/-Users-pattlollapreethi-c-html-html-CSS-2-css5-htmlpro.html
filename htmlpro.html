Data abstraction is one of the most essential and important feature of object oriented programming in C* Abstraction means displaying only essential iformation and luding the details Data abstraction refers to providing only essential information about the data to the outside world, hiding the background details or implementation

Consider a real life example of a man driving a car. The man only knows that pressing the accelerators will mcrease the speed of car or applying brakes will stop the car but he does not know about how on pressing accelerator the speed is actually increasing. he does not knon about the inner mechanism of the cer or the implementation of accelerator, brakes etc in the car.

Abstraction in Header files:One more type of abstraction in C+ can be header files. For example, connder the pow method present in math h header file Whenever we need to calculate power of a number, we simply call the function pow present in the math h header file and pass the numbers as arguments without knowing the underlying algorithm according to which the function is actually caleulating power of numbers.

Advantages of Data Abstraction:

  1. Helps the user to avoid writing the low level code.
  2. Avoids code dupitcation and increases reusability
  3. Can change internal implementation of class independently without affecting the user
  4. Helps to increase security of an application or program as only important detais are provided to the user.
